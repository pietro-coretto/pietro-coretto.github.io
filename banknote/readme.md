# Swiss Banknotes 

Data measuring 6 features on 200 Swiss banknotes (the old-Swiss 1000-franc). The banknotes belong to two classes of equal size: genuine and counterfeit.

* Download: [banknote.RData](raw/main/banknote/banknote.RData)

* Container: R data.frame named  'dat'

* Sample size: 200

* Number of variables: 7




## Variables / Features

* class        classes {genuine, counterfeit}

* Length       Length of bill (mm)

* Left         Width of left edge (mm)

* Right        Width of right edge (mm)

* Bottom       Bottom margin width (mm)

* Top          Top margin width (mm)

* Diagonal     Length of diagonal (mm)
