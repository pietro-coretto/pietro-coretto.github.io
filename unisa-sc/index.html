1. Presentazione del corso

Il corso introduce l'ambiente di calcolo R, e le principali tecniche di programmazione numerica per la statistica. I principali metodi di analisi e visualizzazione dati saranno applicati a casi studio e data set reali. Alla fine del corso lo studente sarà in grado di produrre indicatori, modelli statistici, documenti e report di supporto all'attività decisionale in ambito economico e aziendale. I concetti fondamentali del corso di base di Statistica (II anno) saranno opportunamente richiamati, tuttavia si consiglia di rivedere il materiale.

3. Software e riferimenti bibliografici

I libri di testo, ed altre risorse bibliografiche sono indicati nel syllabus del corso.

Software
Le lezioni si basano sull'uso dell'ambiente di sviluppo R. "R base package" (licenza GPL) è disponibile attraverso il Comprehensive R Archive Network (CRAN) per tutti i principali sistemi oparativi:

  • Linux: seguire le istruzioni per la specifica distro partendo da questa pagina
  • macOS: installare nel seguente ordine a) Development Tools and Libraries; b) XQuartz; c) R base (binary)
  • Microsoft Windows: installare R base (binary)
Si consiglia, inoltre, l'utilizzo di un IDE appropriato. Una delle scelte più popolari è RStudio (licenza AGPL). RStudio è disponibile per i principali sistemi operativi al seguente link (vedi la sezione "Installers for Supported Platforms").

Esistono molti manuali e guide gratuite per R. La documentazione ufficiale è disponibile sul sito del CRAN nella sezione manuali. Prima di considerare manuali avanzati, si consiglia sempre la lettura del manuale introduttivo ufficiale: R-intro.

Altre risorse riguardanti R

  • RDocumentation: una sorta di portale per la documentazione del software
  • R Graph Gallery: estesa collezione di data visualization
  • R-bloggers: blog aggregator di contenuti riguardanti R
  • Awesome R: liste e rankings dei package più utilizzati
