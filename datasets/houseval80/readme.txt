********************************************************************************
DATA SET DESCRIPTION: houseval80.RData
********************************************************************************


Summary
-------
This data set is taken from the 20th  United States Census, conducted by the
Census Bureau (https://www.census.gov/). Data are about house renting.
********************************************************************************


Sampling:..............cross-section
Sample size:...........50 (random subsample of the original census data)
Number of variables....6
Container:.............dat (data.frame)


Variables:
----------
state       US state
region      US standard regional division 
percurban   percentage of the population living in urban area
faminc      median family income in USD
medval      median value of  owner-occupied house in USD
medrent     median rent
