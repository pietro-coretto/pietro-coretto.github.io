********************************************************************************
DATA SET: Breast Cancer Wisconsin (Diagnostic)
********************************************************************************

This is  data set has been obtained at 

http://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+%28Diagnostic%29

and processed to obtain an R data.frame format for teaching purposes.

See link above for details and references.


* Sample size: 569

* Number of variables: 1 outcome variable + 30 features

* Container:  R data.frame named  'dat'

* Variables / Features
  dat[,1]       contains true classes {M = "malignant", B = "Benign"}
  dat[,-1]	30 features obtained from FNA scans
                https://en.wikipedia.org/wiki/Fine-needle_aspiration
